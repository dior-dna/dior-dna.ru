Kinoview

Этим проектом занимаются: Антон

Отображатель расписания кино в Рязани. Предварительно предполагается написать два приложения: android приложение, которое будет показывать расписания кино в Рязани, и веб сайт, к которому будут обращены запросы андроид приложения.

Репозиторий: https://github.com/dior-dna/kinoview

Скетч главного экрана, фильма и кинотеатра

Оригинал изображения

Сервиса

Веб часть хостится тут: http://81.177.160.203:8000/

Данные по фильмам обновляются каждые четыре часа. Реализован сбор данных для кинотеатров: Люксор, Кронверк-Синема

Получить список кинотеатров http://81.177.160.203:8000/service/getCinemas
Получить список фильмов, которые идут в какой-то день http://81.177.160.203:8000/service/getFilms?date=14.03.2013
Получить изображение для фильма. Размер картинки 400 x 225 px. http://81.177.160.203:8000/resources/films/film_1.png
